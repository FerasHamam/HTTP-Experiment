## Getting Started

### Installation

1. generate self-signed SSL certificate

   ```sh
   openssl req -x509 -newkey rsa:4096 -keyout server.key -out server.crt -sha256 -days 365
   ```
2. Install packages
   ```sh
   npm install
   ```
3. Run the server
   ```sh
   npm start
   ```
